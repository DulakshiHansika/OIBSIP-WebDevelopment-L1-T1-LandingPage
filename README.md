# 🎓 TechLearn Academy – Online Learning Platform

A modern and fully responsive landing page designed for online education platforms. **TechLearn Academy** provides an engaging user experience with a clean interface, smooth animations, and well-structured sections to showcase courses, services, and student feedback.

## 🌟 Features
- 🎨 Modern Hero Section with animated floating image effects
- 📱 Fully Responsive Design for desktop, tablet, and mobile devices
- 🧭 Smooth Navigation with seamless scrolling between sections
- 📚 Featured Courses section with attractive course cards
- ⭐ Student Testimonials with interactive hover effects
- 👨‍🏫 About Academy section introducing the learning platform
- 📞 Contact Section with quick contact information
- 🌐 Social Media Links for easy connectivity
- 🎯 Clean UI/UX Design using modern web design principles

## 🛠️ Technologies Used
- HTML5 – Semantic and well-structured web pages
- CSS3 – Flexbox, CSS Grid, Variables, Animations & Responsive Design
- Font Awesome 6 – Professional icons
- Google Fonts (Poppins) – Modern typography
