🎓 TechLearn Academy - Online Learning Platform

TechLearn Academy is a modern, fully responsive landing page template built for online education platforms. It features an interactive layout, animated hero section, clean course catalogs, and structured contact blocks.

✨ Features
Modern Hero Section:  Features an organically shaped, floating, and pulsing background image layout using CSS animations.
Fully Responsive: Perfectly optimized for desktops, tablets, and smartphones.
Clean Navigation: Smooth scroll effects to move seamlessly across different page sections.
Student Testimonials: Elegant review cards with hover effects.
Information Hub: Detailed course layouts, dynamic social icons, and quick contact boxes.

🛠️ Technologies Used
HTML5: Structured, semantic layout markup.
CSS3: Custom properties (variables), Flexbox, grid properties, and keyframe animations.
Font Awesome (v6.5.1):  High-quality vector icons for UI elements.
Google Fonts: "Poppins" typography for a premium, clean design.
